# Renewable Energy Output Prediction System

This project predicts renewable energy output using Machine Learning
based on environmental factors like temperature, wind speed, and solar irradiance.

## Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Streamlit
- Pandas & NumPy

## Features
- Predict energy output
- User-friendly web interface
- Trained ML regression model

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
